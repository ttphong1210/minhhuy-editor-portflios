# 🎯 Tên Dự Án

Website giới thiệu cá nhân / portfolio được xây dựng bằng Vue.js + Laravel.

## 🚀 Tính năng

- Đa ngôn ngữ (vi/en)
- Giao diện responsive
- Trang portfolio, about, contact
- Quản trị backend (Laravel)

## 🔧 Công nghệ

- Frontend: Vue 3 + TailwindCSS
- Backend: Laravel 10
- Cơ sở dữ liệu: MySQL
- Triển khai: GitHub Actions + FTP
